  \begin{columns}
    \begin{column}{.50\textwidth}
      \begin{itemize}
        \item Le dipendenze tra test sono dichiarate \alert{esplicitamente} e risolte \alert{topologicamente}; nessun test può essere eseguito prima dei suoi prerequisiti

        \item Senza DAG, i test RBAC eseguiti prima di quelli di autenticazione producono errori non deterministici invece di fallimenti puliti
      \end{itemize}
    \end{column}

    \begin{column}{.47\textwidth}
      \textbf{Exit codes semantici per CI/CD}
      \vspace{0.5em}
      \begin{itemize}
        \item \textcolor{green!55!black}{\textbf{\texttt{0} CLEAN}} — tutti PASS/SKIP: pipeline prosegue

        \item \textcolor{red!65!black}{\textbf{\texttt{1} FAIL}} — almeno un FAIL: blocco merge/deploy

        \item \textcolor{orange!75!black}{\textbf{\texttt{2} ERROR}} — malfunzionamento interno: revisione manuale

        \item \textcolor{gray!70!black}{\textbf{\texttt{10} INFRA}} — errore infrastrutturale: diagnosi deployment
      \end{itemize}
    \end{column}
  \end{columns}
\end{frame}
