\begin{frame}{Specified oracles: il criterio di valutazione}
  \begin{itemize}
    \item Nessun criterio sistematico per giudicare se un \alert{comportamento osservato} è una violazione

    \item \textbf{Specified oracles}: criterio definito \textit{a priori} dalla \alert{conoscenza del dominio} del test e codificato nella sua logica

    \item Fonti eterogenee, dipendenti dal controllo: \textit{es.} schema OAS, standard tecnici, o costruzioni ad-hoc
  \end{itemize}
\end{frame}
